# ichir0roie-memo

出先や外のPCからかんたんに自分のローカル情報を取り出したり、
保存したりしたい。

+ Webブラウザからアクセスして入出力できる
+ ログイン機能
+ かんたんなURL
  + http://ichir0roie.com/memo
  + WordPress消すか。
  + ssl認証
    + httpsにしたくね。
    + 硬い設定のところから入ると
      + eer_ssl_protocol_error

## テキスト入力機能

+ マークダウン形式で入力した情報を保存する。
+ リアルタイムでコンパイル結果を表示したい。
+ 保存したデータは最終的にGDに送ったりしたい。
